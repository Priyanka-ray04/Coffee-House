# ☕ Bean & Brew Coffee House

A modern and responsive coffee shop website built using **HTML, CSS, and JavaScript**.  
Bean & Brew provides a warm and elegant coffee-themed experience with attractive layouts, responsive design, and interactive elements.

---

## 🌐 Live Demo

🔗 **Live Website:** https://bean-n-brew.netlify.app/

---

## 📖 About The Project

Bean & Brew Coffee House is a front-end project created to practice and improve web development skills. The project focuses on building an attractive user interface with a cozy coffee-shop theme and responsive design principles.

The website includes:

- Homepage
- About Page
- Menu Page
  
---

## ✨ Features

✔ Responsive design for desktop, tablet, and mobile devices  
✔ Sticky navigation bar  
✔ Elegant coffee-themed interface  
✔ Hero sections with attractive background images  
✔ About page with animated statistics counters  
✔ Beautiful menu cards layout  
✔ Interactive hover effects  
✔ Footer with contact details and opening hours  
✔ Clean and organized design structure  
✔ User-friendly experience  

---

## 🛠 Technologies Used

## 🚀 Installation & Usage

Clone the repository:

```bash
git clone https://github.com/Priyanka-ray04/Bean-Brew-Coffee-Shop.git
```

Move into the project folder:

```bash
cd Bean-Brew-Coffee-Shop
```

Open the project:

```bash
Open index.html in your browser
```
